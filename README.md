## Run Locally

Clone the project

```bash
  git clone https://github.com/ainistaranda/landing-page-styled-components.git
```

Go to the project directory

```bash
  cd landing-page-styled-components
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
